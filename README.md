# Sistema de Órdenes de Servicio y Seguimiento de Prendas

## Descripción del proyecto

Aplicación web desarrollada para gestionar órdenes de servicio de una lavandería, permitiendo registrar clientes, controlar prendas, realizar seguimiento del estado de las órdenes y generar reportes básicos.

El sistema busca mejorar la organización del proceso de recepción, lavado y entrega de prendas.

## Objetivo

Desarrollar una solución web que permita administrar eficientemente los servicios de lavandería mediante un sistema de órdenes y seguimiento.

## Funcionalidades principales

### 1. Registro de clientes
- Crear clientes.
- Guardar información de contacto.
- Consultar clientes registrados.

### 2. Registro de órdenes
- Crear órdenes de servicio.
- Asociar clientes.
- Registrar tipo de servicio y precio.

### 3. Registro de prendas
- Registrar prendas ingresadas.
- Guardar cantidad y características.

### 4. Seguimiento de órdenes
- Actualizar estado de la prenda:
  - Recibida.
  - En lavado.
  - Secado.
  - Planchado.
  - Lista para entregar.
  - Entregada.

### 5. Reportes
- Total de clientes.
- Total de órdenes.
- Total de prendas registradas.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## Instalación

1. Descargar el proyecto.
2. Abrir el archivo index.html en un navegador web.
